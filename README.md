## Intro
: &nbsp; 해당 프로젝트는 자바 스프링부트를 이용한 물류 중계 업체의 컴플레인 관리 서비스 API로 <br>
고객사와 배송사 사이에 발생한 문제점을 접수하고 처리하는 프로세스를 담은 스프링부트 프로젝트입니다.

## Project Summary
- 개발언어
  - Java 17
- 프레임워크
  - Spring Boot 3.1.5
- 데이터베이스
  - MySQL 8.0.2
- ORM
  - JPA
- 빌드
  - Gradle / Groovy
